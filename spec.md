# 江苏中医在线 - 用户端需求规格说明书

## 1. 项目概述

### 1.1 项目名称
江苏中医在线用户端 (jsmedicine-user-frontend)

### 1.2 项目类型
中医在线学习平台 - Web端用户界面

### 1.3 核心功能概述
面向中医学习用户的在线教育平台，提供课程学习、医师咨询、在线考核、知识库浏览等功能。

### 1.4 目标用户
- 中医专业学生和从业者
- 中医爱好者
- 需要继续教育的医护人员

## 4. 设计规范

### 4.1 配色方案
| 颜色用途 | 色值 | 说明 |
|---------|------|------|
| 主色调 | #d4a574 | 琥珀色，温暖的中医感 |
| 强调色 | #c19660 | 深琥珀色 |
| 深色 | #92400e | 用于hover和强调 |
| 背景色 | #f5f5f5 | 浅灰背景 |
| 文字色 | #333333 | 正文文字 |
| 辅助色 | #10b981 | 成功/在线状态 |

### 4.2 布局规范
- 最大宽度：1280px (max-w-7xl)
- 栅格系统：12列布局
- 卡片圆角：12px
- 阴影：轻阴影 (shadow-sm)
- 固定头部高度：64px

### 4.3 响应式断点
- 桌面端：≥1280px
- 笔记本端：≥1024px
- 平板端：≥768px
